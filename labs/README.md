Ο τρέχων φάκελος περιέχει τα αρχεία που δημιουργήθηκαν κατά τα εργαστήρια του μαθήματος "Επιστήμη των Δεδομένων".
Είναι σε γλώσσα Matlab/Octave.

**Tα αρχεία .m (MATLAB File Extension), υποστηρίζονται και μπορούν να αναγνωριστούν από το πρόγραμμα GUI Octave.**

Ο φάκελος "drafts" θα χρησιμοποιηθεί ως backup folder, για πρόχειρους κώδικες. Οι οποίοι, στη συνέχεια, θα διορθωθούν.
